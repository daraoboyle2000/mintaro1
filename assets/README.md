# Asset organization

Place generated design assets under this `assets/` directory using the following folders:

- `assets/brand/` — logos, wordmarks, and brand marks.
- `assets/icons/` — reusable UI icon files.
- `assets/illustrations/` — larger decorative or explanatory illustrations.
- `assets/avatars/` — default/profile avatar images.
- `assets/placeholders/` — placeholder images and empty-state graphics.
- `assets/onboarding/` — onboarding screens/graphics.

Each folder includes a `.gitkeep` so the structure is preserved in Git even before files are added.
